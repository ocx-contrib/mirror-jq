# jq

jq is a lightweight, flexible command-line JSON processor. It applies a
concise, expressive filter language to slice, filter, map, and transform
structured data — the way `sed`, `awk`, and `grep` work for plain text. It
reads JSON from stdin or files, evaluates a program against it, and streams
the result to stdout, making it a staple of shell pipelines and CI scripts.

## What's included

- **jq** — the command-line JSON processor (`-n`, `-r`, `-c`, `-s`, filter
  programs, …)
